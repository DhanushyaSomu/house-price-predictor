# Task 1: House Price Predictor (Linear Regression)

AI/ML Internship Task at Maincrafts Technology.

## Objective
Built and evaluated a Linear Regression model to predict median house prices
using the California Housing dataset (built into scikit-learn).

## Tools Used
Python, pandas, NumPy, scikit-learn, matplotlib, seaborn, Jupyter Notebook

## Workflow
1. Loaded the California Housing dataset
2. Performed EDA (distributions, correlations, visualizations)
3. Split data into train/test sets
4. Trained a Linear Regression model
5. Evaluated using MAE, RMSE, R² Score
6. Visualized predictions vs actual values

## Results
- MAE: 0.533
- RMSE: 0.746
- R² Score: 0.576
